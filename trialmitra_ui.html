🏗️ System Architecture

TrialMitra implements a federated learning architecture where:

  • Each hospital maintains its own local patient dataset
  • Local models are trained without sharing raw data
  • Only model parameters are shared with the central server
  • Global model is created through federated averaging (FedAvg)
  • Chatbot interactions enhance prediction features
